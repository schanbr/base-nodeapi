### Configurações

- ESlint - CommonJS e Google Standard
- Express
- Arquitetura MVC no projeto
- Routes.js
- Nodemon

### Instalar

1. Clone o repositório
1. Instale as dependências
   "npm install" ou "yarn install"
1. Execute o projeto
   "npm run start" ou "yarn start"
